BlogApp Backend
A simple backend application for a blogging platform, designed to help developers practice and learn backend development concepts. This project includes features like creating posts, adding comments, and liking posts, making it a great starting point for understanding RESTful APIs, database integration, and server-side logic.

Features.......
Create Posts: Add new blog posts with a title and body.
Add Comments: Comment on existing posts.
Like Posts: Like posts and track the number of likes.
Database Integration: Uses MongoDB for data storage.
RESTful API: Follows REST principles for API design.


Technologies Used.......
Node.js: Backend runtime environment.
Express.js: Web framework for building APIs.
MongoDB: NoSQL database for storing posts, comments, and likes.
Mongoose: ODM library for MongoDB.
